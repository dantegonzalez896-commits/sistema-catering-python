# Sistema de Gestión para Catering

## Descripción
Sistema desarrollado en Python enfocado en la administración de pedidos y gestión operativa para negocios de comida. El programa permite una navegación estructurada mediante submenús interactivos y facilita la toma de decisiones al permitir la exportación de la información directamente a Microsoft Excel.

## Características Principales
* Interfaz de consola con navegación por submenús para fácil acceso a las funciones.
* Gestión y almacenamiento estructurado de datos.
* Exportación automatizada de registros y reportes hacia hojas de cálculo Excel.

## Tecnologías Utilizadas
* Python 3.x
* SQLite3 / JSON (para el manejo de base de datos o diccionarios)
* Librerías: sqlite3, datetime, openpyxl

## Instrucciones de Instalación y Ejecución
1. Clonar el repositorio localmente.
2. Instalar las dependencias necesarias mediante el comando:
   `pip install -r requirements.txt`
3. Ejecutar el archivo principal del sistema:
   `python main.py`
